# Radar-and-Remote-Sensing-Datasets
Consiste em transcrições de aulas e artigos técnicos focados em geotecnologias aplicadas ao monitoramento ambiental e análise de dados espaciais. Explora os fundamentos de radar de abertura sintética (SAR) e o processamento de imagens para o estudo da vegetação e desmatamento com tutoriais de Python e Google Earth Engine.
